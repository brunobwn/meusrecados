# CRUD de recados feito para trabalho de final de módulo na Growdev

O sistema de recados é composto por um conjunto de funcionalidades CRUD (criar, ler, atualizar, excluir) que permitem ao usuário realizar o cadastro, efetuar login e gerenciar recados vinculados à sua conta. Os recados são armazenados no localStorage do navegador e a aplicação foi desenvolvida inteiramente em JavaScript e utiliza o framework Bootstrap para o estilo visual. Além disso, foram implementadas funcionalidades adicionais, como paginação dos registros e a opção de escolha do número de registros exibidos por página.

Deploy: [GitPages](https://brunobwn.github.io/meusrecados/)
